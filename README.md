# True Contact — AI Companion self-check

A 2-minute self-assessment that scores AI-companion over-attachment and routes the visitor
into the free 7-Day Unplug email course. Static page; the capture form posts to a Cloudflare
Worker that writes the lead (name, email, age, sex, quiz type/score/band) into MailerLite.

Live: https://yanivro.github.io/true-contact-unplug/
